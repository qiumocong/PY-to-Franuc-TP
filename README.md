# PY-to-Franuc-TP

利用python语言实现Fanuc机器人的示教器程序编写  
Use Python language to implement Fanuc robot teach pendant program writing

一个完整的Python库，用于生成Fanuc机器人TP（示教器）程序，包含所有主要的示教器程序命令。  
A complete Python library for generating Fanuc robot TP (Teach Pendant) programs, including all major teach pendant programming commands.

## 功能特性 / Features

- ✅ **完整的TP命令支持** / Complete TP command support
  - 运动命令 (J, L, C) / Movement commands (Joint, Linear, Circular)
  - 逻辑控制 (IF/ELSE, FOR, WHILE) / Logic control
  - I/O操作 (DO, DI, RO, RI, GO, GI) / I/O operations
  - 等待命令 (WAIT, PAUSE, STOP) / Wait commands
  - 程序调用 (CALL, RUN, ABORT) / Program calls
  - 标签和跳转 (LBL, JMP) / Labels and jumps

- ✅ **位置管理** / Position management
  - 笛卡尔坐标位置 / Cartesian coordinate positions
  - 关节角度位置 / Joint angle positions
  - 自动格式化为TP格式 / Automatic formatting to TP format

- ✅ **链式调用API** / Fluent API with method chaining
- ✅ **自动生成标准TP程序格式** / Automatic standard TP program format generation
- ✅ **类型安全和验证** / Type safety and validation

## 快速开始 / Quick Start

### 基本示例 / Basic Example

```python
from fanuc_tp import FanucTPWriter, Position

# 创建TP程序写入器 / Create TP writer
tp = FanucTPWriter("DEMO")

# 定义位置 / Define positions
home_pos = Position.from_joints(0, 0, 0, 0, 0, 0)
work_pos = Position.from_xyz(100, 200, 300, 0, 180, 0)

# 添加位置到程序 / Add positions to program
tp.add_position(1, home_pos)
tp.add_position(2, work_pos)

# 创建程序 / Create program
tp.add_comment("Demo Program") \
  .move_joint(1, 50) \
  .move_linear(2, 100) \
  .set_digital_output(1, True) \
  .wait_time(1.0) \
  .set_digital_output(1, False) \
  .move_joint(1, 75)

# 生成TP程序 / Generate TP program
program = tp.generate_tp_program()
print(program)

# 保存到文件 / Save to file
tp.save_to_file("demo.ls")
```

## 主要API / Main API

### FanucTPWriter 类 / Class

#### 运动命令 / Movement Commands
- `move_joint(position, speed, termination)` - 关节运动 / Joint movement
- `move_linear(position, speed, termination)` - 直线运动 / Linear movement  
- `move_circular(position, speed, termination)` - 圆弧运动 / Circular movement

#### 逻辑控制 / Logic Control
- `if_condition(condition)` - IF语句 / IF statement
- `else_statement()` - ELSE语句 / ELSE statement
- `endif()` - ENDIF语句 / ENDIF statement
- `for_loop(register, start, end)` - FOR循环 / FOR loop
- `while_loop(condition)` - WHILE循环 / WHILE loop

#### I/O 操作 / I/O Operations
- `set_digital_output(number, value)` - 设置数字输出 / Set digital output
- `set_robot_output(number, value)` - 设置机器人输出 / Set robot output
- `set_group_output(number, value)` - 设置组输出 / Set group output

#### 等待和控制 / Wait and Control
- `wait_for_condition(condition)` - 等待条件 / Wait for condition
- `wait_time(seconds)` - 等待时间 / Wait for time
- `pause()` - 暂停 / Pause
- `stop()` - 停止 / Stop

#### 程序控制 / Program Control
- `call_program(name)` - 调用程序 / Call program
- `label(number)` - 标签 / Label
- `jump_to_label(number, condition)` - 跳转到标签 / Jump to label

### Position 类 / Class

```python
# 笛卡尔坐标 / Cartesian coordinates
pos1 = Position.from_xyz(x, y, z, w, p, r)

# 关节角度 / Joint angles  
pos2 = Position.from_joints(j1, j2, j3, j4, j5, j6)
```
